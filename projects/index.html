Large Number Arithmetic Operations#

IIT Gandhinagar, Jan 2024 - Present
Individual Contributor, M.Tech. Thesis Work
Advisor: Prof. Abhishek Bichhawat, Co-advisor: Prof. Yuvraj Patel (The University of Edinburgh)
(Project Link)

  • Developed optimized algorithms for large number arithmetic to enhance cryptographic encryption.
  • Leveraged system-level features such as Transparent Huge Pages, Advanced Vector Extensions, and Caching.
  • Utilized tools like PERF Events and Valgrind for performance analysis.
  • Implemented and benchmarked Vedic mathematical algorithms (e.g., Urdhva Tiryagbhyam) against libraries like GNU Multiprecision Library.

TennisServe: A Parallel Game Matching Server with OpenMP & MPI#

IIT Gandhinagar, Jan 2024 - April 2024
Individual Contributor
(Project Link)
